¿Cómo funciona?

Este dashboard es una simulación para entender el flujo de trabajo en un entorno de soporte técnico. Aunque no tiene backend real, demuestra mi capacidad para:

  • Crear interfaces útiles para usuarios no técnicos
  • Organizar información de manera clara y accesible
  • Resolver problemas con herramientas básicas (HTML/CSS/JS)
