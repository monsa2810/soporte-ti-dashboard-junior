# Contribuciones

Este proyecto es un ejercicio personal para demostrar habilidades básicas en desarrollo frontend. No acepto contribuciones externas, pero agradezco cualquier feedback en forma de issues.

## ¿Cómo usar este proyecto?
1. Clona el repositorio: `git clone https://github.com/tu-usuario/soporte-ti-dashboard-junior.git`
2. Abre `index.html` en tu navegador
3. Explora las funcionalidades básicas del dashboard

## ¿Por qué este proyecto?
Como buscador de primer empleo en TI, quiero mostrar mi capacidad para:
- Aplicar conocimientos teóricos en proyectos prácticos
- Crear soluciones simples pero efectivas
- Documentar mi trabajo de manera clara
