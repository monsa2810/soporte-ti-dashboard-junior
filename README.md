# Dashboard de Soporte Técnico Junior

Proyecto personal para simular un sistema básico de gestión de tickets de soporte técnico. Diseñado para practicar desarrollo frontend y entender flujos operativos en help desk.

## 🌟 ¿Por qué este proyecto?

Como buscador de primer empleo en TI, creé este proyecto para demostrar mi capacidad de:
- Aplicar conocimientos teóricos en proyectos prácticos
- Resolver problemas técnicos reales con herramientas básicas
- Documentar mi trabajo de manera clara y profesional
- Aprender y mejorar continuamente

## 🛠️ Funcionalidades clave

- **Sistema de gestión de tickets** con estado, prioridad y descripción
- **Filtrado dinámico por prioridad** (Alta/Media/Baja)
- **Formulario de registro de nuevos tickets**
- **Estadísticas visuales** de resolución y estado
- **Interfaz responsive** optimizada para móviles y escritorio
- **Sistema de autenticación básica** (token: `soporte-junior`)
- **Actualización de estadísticas en tiempo real**

## 💻 Tecnologías utilizadas

- **HTML5**: Estructura semántica del dashboard
- **CSS3**: Estilos modernos con variables CSS y diseño responsive
- **JavaScript (vanilla)**: Funcionalidad de filtrado, registro y autenticación
- **GitHub Pages**: Hosting para demostración
- **Git**: Control de versiones
