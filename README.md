# Dashboard de Soporte Técnico Junior

Proyecto personal para simular un sistema básico de gestión de tickets de soporte técnico. Diseñado para practicar desarrollo frontend y entender flujos operativos en help desk.

## 🛠️ Mejoras recientes
- **Sistema de filtrado por prioridad** (Alta/Media/Baja)
- **Formulario de registro de nuevos tickets**
- **Interfaz responsive mejorada** con efectos visuales sutiles
- **Documentación técnica detallada** de cada componente

## 💻 Tecnologías
- HTML5, CSS3, JavaScript (vanilla)
- Sin backend ni base de datos — 100% frontend

## 📌 Nota
Como buscador de primer empleo en TI, creé este proyecto para demostrar mi capacidad de resolver problemas técnicos reales con herramientas básicas. Es educativo y funcional — ideal para roles junior en soporte o desarrollo.

🔗 [Ver demo en GitHub Pages](https://monsa2810.github.io/soporte-ti-dashboard-junior)
